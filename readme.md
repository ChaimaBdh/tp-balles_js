## Préalable 

Télécharger et extraire l'archive du projet ou cloner ce dépôt :  

```bash
git@github.com:ChaimaBdh/tp-balles_js.git
```

## Installation des fichiers

Dans le répertoire du projet, exécuter la commande suivante :

```bash
npm install
```
  
  
## Execution du projet 

```bash
npm run dev-server
```
